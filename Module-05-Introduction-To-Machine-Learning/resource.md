আজকে আপনারা শুরু করবেন Module 05: Introduction to Machine Learning। এই মডিউল দেখে আপনারা Machine Learning সম্পর্কে ভালো একটি ধারনা পেয়ে যাবেন। আজকের মডিউলটি পুরোপুরি থিওরিকাল হবে, যেইগুলো রিয়েল লাইফ উদাহরণ দিয়ে বুঝানো হয়েছে। থিওরিকাল মডিউল হওয়া বেশি মনোযোগসহকারে মডিউলটি দেখবেন।



মডিউলের শেষে একটি Quiz থাকবে, যা দিয়ে আপনারা শেখা বিষয়গুলো যাচাই করতে পারবেন। এরপর থাকবে Quiz Explanation, যেখানে Quiz এর প্রতিটি প্রশ্নের ব্যাখ্যা পেয়ে যাবেন। নিয়মিত কোড প্র্যাকটিস করেন ও মডিউলের সব অংশ সম্পূর্ণ করেন, এতে আপনার দক্ষতা আরও বেড়ে যাবে।
